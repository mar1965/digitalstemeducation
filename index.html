We are a social enterprise

Digital STEM Education (or DSE) is a social enterprise. Our primary goal is to provide a stimulating learning environment that gets students excited about science, technology, engineering and mathematics fields through a complete STEM learning package.

Our vision? Empower America's next generation of engineers!

Learn More!

Integrated Learning Package

The DSE learning package is integrated into 3 major components:

  • Hardware kit
  • Software
  • Project curriculum (self-guided documentation)

What's inside the toolkit?

  • 1 STEMSEL re-usable micro controller board
  • 1 Buzzer
  • 3 LEDs (red, green, yellow)
  • 1 Push Button
  • 1 DC motor with fan
  • 2 Light Sensors
  • 2 Temp sensors
  • 1 Potentiometer with knob
  • 1 (3xAA) Battery holder (Batteries not included)
  • 1 USB cable

A simple process that emphasizes PBL (Problem Based Learning)

  • Design using the circuit designer
  • Assemble using the hardware kit
  • Program using the software

Get Started!

Get Started!

It's easy to get started with Digital STEM...

Simply download the ezSystem setup file and run the installer. If you find that you are having trouble installing ezSystem, you may also need to install the required version of Microsoft .NET Framework 4.0. After the installation, you can start any of the project files listed, most of which are also accompanied by an interactive tutorial.
